## ログイン方法

### 初回ログイン

1. アプリケーションURL（https://your-domain.com）にアクセス
2. 「Googleでログイン」ボタンをクリック
3. Googleアカウントを選択して認証
4. 初回は「承認待ち」画面が表示されます
5. 管理者による承認後、運営ダッシュボードにアクセス可能

### 通常ログイン

1. アプリケーションURLにアクセス
2. 「Googleでログイン」ボタンをクリック
3. 承認済みの場合、自動的に運営ダッシュボードにリダイレクト

## ダッシュボードの見方

運営ダッシュボードでは、受講生の状況を一目で確認できます。

### KPIカード

| 項目 | 説明 |
|------|------|
| 総受講生数 | 登録済みの全受講生数（承認済み/未承認の内訳） |
| アクティブ受講生 | 過去7日間に学習活動があった受講生数 |
| 未確認課題 | フィードバックが未送信の課題提出数 |
| 平均進捗率 | 全受講生の平均学習進捗率 |

### グラフ

- **進捗分布**: 受講生の進捗率の分布を表示
- **提出トレンド**: 直近の課題提出数の推移

### 最近の未確認課題

確認が必要な課題の一覧が表示されます。クリックして詳細を確認できます。

## 受講生管理

### 受講生一覧の確認

1. サイドバーの「受講生管理」をクリック
2. 登録済みの全受講生が一覧表示されます

### 検索・フィルター

- **検索**: 名前やメールアドレスで検索
- **ステータスフィルター**: 承認済み/未承認でフィルタリング
- **進捗フィルター**: 進捗率でフィルタリング

### 受講生の詳細確認

1. 受講生一覧から対象者をクリック
2. 詳細画面で以下を確認：
   - 基本情報（名前、メール、登録日）
   - 学習進捗（全体進捗率、Phase別進捗）
   - 課題提出履歴
   - 最終学習日時

### 受講生の承認

1. サイドバーの「承認管理」をクリック
2. 承認待ちの受講生一覧が表示されます
3. 「承認」または「却下」ボタンをクリック
4. 確認ダイアログで操作を確定

## 課題管理

### 課題一覧の確認

1. サイドバーの「課題管理」をクリック
2. 全ての課題提出が一覧表示されます

### フィルター機能

- **ステータス**: 未確認/確認済みでフィルタリング
- **コンテンツ**: 特定のコンテンツの課題のみ表示
- **期間**: 提出日時で絞り込み

### フィードバックの送信

1. 課題一覧から対象の課題をクリック
2. 提出内容を確認（コードまたはURL）
3. フィードバック入力欄にコメントを記入
4. 「フィードバックを送信」ボタンをクリック

### フィードバックのポイント

- 具体的な改善点を指摘
- 良かった点も必ず言及
- 次のステップを提案
- 参考リソースがあれば共有

## コンテンツ管理

### コンテンツ構造

```
Phase（フェーズ）
└── Week（週）
    └── Content（コンテンツ）
        - video: 動画コンテンツ
        - text: テキスト教材
        - exercise: 演習問題
```

### コンテンツの確認

1. サイドバーの「コンテンツ管理」をクリック
2. Phase > Week > Content の階層で表示
3. 各コンテンツをクリックして詳細を確認

### コンテンツの編集

1. 編集したいコンテンツの「編集」ボタンをクリック
2. 必要な項目を更新：
   - タイトル
   - 説明
   - コンテンツ本体（URL、Markdownテキスト等）
3. 「保存」ボタンで更新

### コンテンツの追加

1. 追加先のWeekを選択
2. 「コンテンツを追加」ボタンをクリック
3. コンテンツタイプを選択
4. 必要情報を入力
5. 「作成」ボタンで追加

## お知らせ管理

### お知らせ一覧

1. サイドバーの「お知らせ管理」をクリック
2. 全てのお知らせが一覧表示されます

### お知らせの作成

1. 「新規作成」ボタンをクリック
2. 以下を入力：
   - タイトル（必須）
   - 本文（Markdown形式対応）
   - 公開日時（即時公開または予約投稿）
3. 「公開」または「下書き保存」をクリック

### お知らせの編集・削除

- 編集: 対象のお知らせの「編集」ボタンをクリック
- 削除: 対象のお知らせの「削除」ボタンをクリック（確認あり）

## よくある操作

### 新規受講生の承認フロー

1. 新規登録の通知を受け取る
2. 「承認管理」画面で申請者を確認
3. 必要に応じて申請者に連絡
4. 「承認」ボタンで承認完了
5. 受講生に承認完了を通知

### 課題のフィードバックフロー

1. 「未確認課題」の通知を確認
2. 「課題管理」画面で詳細を確認
3. 提出内容をレビュー
4. フィードバックを記入して送信
5. 必要に応じてフォローアップ

### お知らせの定期投稿

1. 週次/月次のお知らせを事前に作成
2. 公開日時を設定して予約投稿
3. 自動的に指定日時に公開
